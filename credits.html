Hey there!

MmmDot was a game designed and developed by students of CS4436 - Selected Topics (Game Audio Development) at Western University.

MmmDot was made using the p5.js and p5.collide2D libraries. p5.collide2D contains some versions of, and references to, the functions in Jeffrey Thompson's Collision Detection Book.

His code is CC BY-NC-SA 4.0, so, this is too!

The source sounds for this game are provided from the Apple Sound Library
